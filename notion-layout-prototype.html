topbar · h=44 (frame 너비만 덮음)

frame / page content

본문 max-width 720 · 좌우 자동 margin · 독서 가독성 우선

Notion 웹은 media query 가 아니라 JS 가 window.innerWidth 로 판단해서 .layout 클래스를 런타임 swap 합니다 (desktop ↔ tablet ↔ phone ↔ side-peek ↔ center-peek). 본문 폭은 기본 모드에서만 max 720 으로 고정, peek 열림 상태에서는 1fr 로 풀려 frame 이 줄어드는 만큼 content 도 따라 줄어듭니다.

사이드바는 기본 270, 최소 270, 최대 600. 사용자가 드래그로 폭을 조정할 수 있으며 키보드 화살표로도 조정 가능합니다 (role="separator"). 접힘 상태에서는 햄버거 버튼이 표시되고, 호버 시 floating drawer로 프리뷰, 클릭 시 inline 복원됩니다.

Side peek은 기본 651.5, 최소 564, 최대 1710의 드로어입니다. 메인 프레임 위에 오른쪽에서 슬라이드로 열리며, 사이드바와 상호 조정 규칙이 있습니다 — 조정 중인 주체가 우선이며 상대는 자동으로 접히거나 닫힙니다.

Library panel은 "더 보기" 진입 시 사이드바 우측에서 나오는 제3의 드로어입니다. 폭 320, URL path 라우팅 기반이며 peek와 상호 배타적입니다. 사이드바 z-index보다 낮아 사이드바 밑에서 슬라이드 나오는 시각 효과를 가집니다.

Center peek은 모달 형태로 화면 중앙에 뜨며 backdrop dim이 적용됩니다. 내부 콘텐츠 폭은 960이고, center-peek-with-layout-panel 변형은 우측 sub-panel(280px)을 동시에 표시합니다.
